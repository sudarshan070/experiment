Open a file:
```
vi filename
```

To go into edit mode:
```press ESC and type I
```

To go into command mode:

```press ESC
```

To save a file
```
press ESC and type :w fileName
```

To save a file and quit:
```
press ESC and type :wq
```

for run script 


```
./filename
```
OR 
```
chmod +x filename
./filename
```
